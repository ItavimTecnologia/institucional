# ITAVIM Tecnologia

Site institucional da **ITAVIM Tecnologia**, empresa especializada em soluções de TI para empresas que buscam crescer com segurança, eficiência e inovação.

## Sobre

A ITAVIM oferece serviços de suporte técnico, infraestrutura em cloud, segurança da informação, consultoria em TI, CFTV IP e TI para advocacia.

## Tecnologias

- HTML5, CSS3 e JavaScript puro
- Fontes: [Sora](https://fonts.google.com/specimen/Sora) e [DM Sans](https://fonts.google.com/specimen/DM+Sans) via Google Fonts
- Deploy via [Surge.sh](https://surge.sh)

## Deploy

O site está publicado em: **[itavim.surge.sh](https://itavim.surge.sh)**

Para atualizar o deploy:

```bash
surge . itavim.surge.sh
```
